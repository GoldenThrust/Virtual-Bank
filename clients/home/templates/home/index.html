About

This project was born from a pivotal moment during my initial project presentation at New Horizon. Witnessing widespread difficulties among individuals in testing payment gateways struck a chord with my own challenges in this area. While Stripe provided a solution, its real payment nature posed limitations even in test mode. This realization sparked the idea to develop an API for transaction simulation, enabling a seamless observation of transaction flows between simulated bank accounts.

The journey commenced as a response to a genuine need in the field rather than a mere academic obligation. The evolution from this initial spark to a tangible solution encapsulates the essence of this project
